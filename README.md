# TaskList AI

TaskList AI — это современное приложение для управления повседневными задачами, созданное на Flutter. Оно не только позволяет планировать и отслеживать задачи, но и интегрирует **AI-помощника**, который помогает оптимизировать расписание, расставлять приоритеты и распределять нагрузку на день.

## Основные возможности

* **Создание и управление задачами**: добавление, редактирование, удаление и группировка задач по категориям и времени.
* **Умное планирование с AI**: AI-помощник анализирует ваши задачи, предлагает оптимальный порядок выполнения и рекомендует перенос задач для более равномерной нагрузки.
* **Напоминания и уведомления**: получение уведомлений о срочных задачах и важных событиях.
* **Категории и приоритеты**: визуальное выделение задач по важности и срочности, цветовые метки и фильтры.
* **История и аналитика**: отслеживание выполненных задач, статистика продуктивности, графики и отчёты.
* **Интерактивный UI**: удобные модальные окна, анимации раскрытия задач, «See More/See Less» и поддержка темной темы.
* **Поддержка AI-чата**: возможность задать AI вопрос по планированию, получить советы по организации времени и рекомендации по выполнению задач.

## Технологии

* **Flutter** для кроссплатформенной мобильной разработки.
* **Dart** для бизнес-логики и интерактивного UI.
* **Интеграция AI** через локальные модели или сторонние API (например, OpenAI) для анализа и оптимизации задач.
* **База данных**: SQLite/NoSQL для хранения задач, категорий и истории.
* **Пуш-уведомления** для напоминаний.

## Пример использования

1. Пользователь добавляет задачи на день.
2. AI-помощник анализирует задачи и предлагает оптимальный порядок выполнения с учётом времени и приоритетов.
3. Пользователь получает уведомления о срочных задачах и рекомендации по плану дня.
4. В конце дня можно посмотреть статистику и эффективность выполнения задач.
